# ax12-arduino
Using Dynamixel AX-12A Servo with Arduino

Wiring Diagram

AX-12A  pin1: GND  --->  DC -12V
AX-12A  pin2: VDD  --->   DC +12V
AX-12A pin3: DATA --->  Tx pin of Arduino

*** Connect  the two grounds together ***
